/api/1080p 🚀 API信息

API Endpoints

  • GET /api/1080p
    Returns the latest Bing wallpaper in 1080p resolution.
  • GET /api/4k
    Returns the latest Bing Wallpaper in 4K resolution.
  • GET /api/json
    Returns a JSON object containing the wallpaper's info.
